# CorePilot — Windows BT Operasyon Asistanı

CorePilot, kurum bilgisayarlarında BT operasyon hızını artırmak için tasarlanmış güçlü bir masaüstü çözümdür. Tek ekranda izleme, bakım ve hızlı müdahale akışlarını birleştirir.

## Neden CorePilot?

- Canlı sistem görünürlüğü (CPU, RAM, Disk, Ağ)
- Kritik durumlarda hızlı aksiyon alma
- Teknik ekibin günlük iş yükünü azaltan operasyon akışı
- Daha düzenli, daha hızlı, daha güvenli BT yönetimi

## Kurulum

1. Bu depodaki [CorePilot-Setup.exe](CorePilot-Setup.exe) dosyasını indirin.
2. Kurulum dosyasını çalıştırın.
3. Sihirbaz adımlarını tamamlayın.
4. Kurulum sonrası uygulamayı Başlat menüsünden açın.

> Ek kurulum gerekmez. Python veya ek paket yüklemeniz istenmez.

## Kullanım

- Dashboard üzerinden sistem durumunu anlık takip edin.
- Bakım/onarım işlemlerini ilgili ekranlardan çalıştırın.
- Uyarı ve önerileri takip ederek performans/süreklilik iyileştirmesi yapın.

## Güncelleme

Yeni sürümlerde sadece güncel `CorePilot-Setup.exe` dosyasını çalıştırmanız yeterlidir.

## Destek

Kurulum veya kullanım sorunu yaşarsanız, uygulamayı Yönetici olarak çalıştırıp tekrar deneyin.
